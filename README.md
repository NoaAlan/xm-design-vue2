  # Xm Design
  基于 vant 和 css 变量的可定制主题的 Vue2 UI 组件库
</div> 

### 使用

启动预览

```
npm run docs:dev
```

打包组件

```
npm run build <组件>
```

### 新增组件

请使用 pr 的方式

```shell
npm run create
```

你可以：

- sfc 语法组件
- 覆盖 vant 组件
- jsx 语法组件

主题样式变量文件 [xmi.theme.less](./xmi.theme.less)
