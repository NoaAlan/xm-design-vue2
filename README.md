  # Xm Design
  基于 vant 和 css 变量的可定制主题的 Vue2 UI 组件库
</div> 

### 安装

```
yarn add @xm/design-vue
```

### 新增组件

输入

```shell
npm run create
```

组件类型 3 种：

- sfc 语法组件
- 覆盖 vant 组件
- jsx 语法组件

主题样式变量文件 [xmi.theme.less](./xmi.theme.less)
