# van-loading

### 使用

```js
import { VanLoading } from '@xm/design-vue'

Vue.use(VanLoading)
```

## 示例

### 普通用法

```html
<van-loading />
<van-loading size="24px">加载中...</van-loading>
<van-loading size="24px" type="spinner">加载中...</van-loading>
```

### 带背景
```html
<van-loading background size="24px">加载中...</van-loading>
<van-loading 
  background 
  size="24px" 
  type="spinner"
>
  加载中...
</van-loading>
```
